# Especificaciones Técnicas — Cisco Catalyst 8200 Series Edge

| Categoría           | Especificación                                      |
| ------------------- | --------------------------------------------------- |
| Tipo de equipo      | Router / Edge Platform SD-WAN                       |
| Modelos             | C8200-1N-4T, C8200L-1N-4T                           |
| Formato             | 1RU                                                 |
| CPU                 | 8 cores (C8200) / 4 cores (C8200L)                  |
| Memoria RAM         | 8 GB (C8200) / 4 GB (C8200L), ampliable hasta 32 GB |
| Almacenamiento      | Flash 8 GB + M.2 (hasta 600 GB NVMe)                |
| Puertos integrados  | 4× 1G (2 RJ45 + 2 SFP)                              |
| Slots               | 1× NIM + 1× PIM                                     |
| Throughput IPsec    | Hasta **1 Gbps** (C8200) / **500 Mbps** (C8200L)    |
| SD-WAN tunnels      | Hasta **2500**                                      |
| NAT sessions        | Hasta **2 millones**                                |
| Rutas IPv4          | Hasta **4 millones**                                |
| Rutas IPv6          | Hasta **4 millones**                                |
| VRFs                | Hasta **4000**                                      |
| Seguridad integrada | NGFW, IPS/IDS, AMP, URL Filtering                   |
| SASE                | Compatible (Umbrella SIG)                           |
| SD-WAN              | Cisco Catalyst SD-WAN                               |
| Routing tradicional | OSPF, BGP, EIGRP, IS-IS, RIP                        |
| QoS                 | QoS avanzado, FEC, AAR                              |
| Virtualización      | Docker / KVM (solo C8200)                           |
| Cellular            | LTE Cat18 y 5G                                      |
| Gestión             | SD-WAN Manager, DNA Center                          |
| PoE                 | Opcional vía módulos NIM                            |
| Consumo típico      | ~35 W                                               |
| MTBF                | ~692.000 horas                                      |
