# Especificaciones Técnicas — Dell PowerEdge R360

| Categoría                          | Especificación                                                                        |
| ---------------------------------- | ------------------------------------------------------------------------------------- |
| Tipo de equipo                     | Servidor rack 1U                                                                      |
| Serie / modelo                     | Dell PowerEdge R360                                                                   |
| Formato                            | 1U rack server                                                                        |
| Procesador                         | 1 x Intel Xeon 6300 series o 1 x Intel Xeon E-2400 series                             |
| Procesador alternativo             | 1 x Intel Pentium G7400 / G7400T                                                      |

| Velocidad de memoria               | Hasta 4400 MT/s                                                                       |
| Controladoras de almacenamiento    | HBA355i, PERC H355, PERC H755, BOSS-N1, S160                                          |
| Almacenamiento frontal             | Hasta 4 x 3.5" SAS/SATA o hasta 8 x 2.5" SAS/SATA                                     |

| Red integrada                      | 2 x 1 GbE LOM                                                                         |
| Gestión                            | iDRAC9, iDRAC Direct, Redfish REST API                                                |
| Software de gestión                | OpenManage Enterprise, CloudIQ, integración con VMware, Microsoft, Ansible, Terraform |
| Seguridad                          | Secure Boot, Silicon Root of Trust, TPM 2.0, Secure Erase, firmware firmado           |
| GPU                                | 1 x GPU single-width de 60 W                                                          |
| Expansión PCIe                     | Hasta 2 slots PCIe Gen4                                                               |
| Fuente de alimentación             | 600 W Platinum o 700 W Titanium, hot-swap redundante                                  |
| Refrigeración                      | Air cooling, hasta 4 ventiladores                                                     |
| Puertos                            | USB 2.0 / 3.2, VGA, serial, puerto iDRAC dedicado                                     |
| Dimensiones                        | 42.8 mm alto × 482.0 mm ancho × 585.3 mm fondo sin bezel / 598.9 mm con bezel         |
| Sistemas operativos soportados     | Ubuntu Server LTS, Windows Server con Hyper-V, RHEL, SUSE, VMware ESXi                |
| Uso recomendado                    | Pyme, ROBO / edge, colaboración, bases de datos ligeras, virtualización               |
