# Especificaciones Técnicas — Supermicro 119U-7

| Categoría                    | Especificación                                                              |
| ---------------------------- | --------------------------------------------------------------------------- |
| Tipo de equipo               | Servidor rack 1U                                                            |
| Modelo / referencia asociada | Supermicro 119U-7 / SuperServer SYS-1028U-TNRTP+                            |
| Formato                      | 1U rackmount                                                                |
| Procesador                   | Doble socket R3 (LGA 2011), compatible con Intel Xeon E5-2600 v4 / v3       |
| Núcleos máximos              | Hasta 22 núcleos por CPU                                                    |
| Chipset                      | Intel C612                                                                  |
| Memoria                      | 24 ranuras DIMM DDR4                                                        |
| Capacidad máxima de memoria  | Hasta 3 TB ECC 3DS LRDIMM; hasta 1.5 TB ECC RDIMM/LRDIMM                    |
| Tipo de memoria              | DDR4 ECC 2400 / 2133 / 1866 / 1600 MHz; soporte NVDIMM 2666 / 2400          |
| Almacenamiento frontal       | 10 bahías hot-swap de 2.5"                                                  |
| Configuración de discos      | 8 puertos SATA3 + 2 puertos híbridos NVMe/SATA3; SAS3 opcional mediante AOC |
| Red integrada                | 2 puertos 10G SFP+                                                          |
| Gestión                      | IPMI 2.0 con KVM y LAN dedicada                                             |
| Expansión PCIe               | 2 x PCIe 3.0 x16 + 2 x PCIe 3.0 x8                                          |
| Fuente de alimentación       | Fuentes redundantes hot-swap de 750 W                                       |
| Eficiencia de fuente         | 80 Plus Platinum                                                            |
| Dimensiones                  | 1U; ancho aprox. 437 mm; fondo aprox. 724 mm                                |
| Uso recomendado              | Virtualización, cargas empresariales, alto rendimiento, almacenamiento NVMe |
