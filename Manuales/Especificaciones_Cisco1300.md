# Especificaciones Técnicas — Cisco Catalyst 1300 / 1300X Series

| Categoría              | Especificación                                          |
| ---------------------- | ------------------------------------------------------- |
| Tipo de equipo         | Switch gestionado L3 (SMB y branch offices)             |
| Series                 | Catalyst 1300 y Catalyst 1300X                          |
| Sistema operativo      | Linux personalizado Cisco                               |
| Gestión                | Cisco Business Dashboard, app móvil, Web GUI, CLI, SNMP |
| Capas                  | Layer 2 y Layer 3                                       |
| Routing dinámico       | RIP v2 (1300), RIP + OSPF v2/v3 (1300X)                 |
| Capacidad de switching | Hasta **480 Gbps**                                      |
| Forwarding rate        | Hasta **357 Mpps**                                      |
| VLANs                  | Hasta **4094 VLANs**                                    |
| Stacking               | Hasta **8 switches** por stack                          |
| Stacking bandwidth     | Mayor ancho de banda en modelos 1300X                   |
| Puertos                | 1G, 2.5G, 5G, 10G, 25G (según modelo)                   |
| PoE                    | PoE (802.3af), PoE+ (802.3at), PoE++ (802.3bt 60W)      |
| Presupuesto PoE        | Hasta **740 W**                                         |
| Multigigabit           | Soportado (2.5G y 5G)                                   |
| Jumbo frames           | Hasta **9000 bytes**                                    |
| Tabla MAC              | Hasta **32.000 entradas**                               |
| Seguridad L2/L3        | 802.1X, DHCP Snooping, DAI, IPSG, ACLs                  |
| IPv6                   | Soporte completo IPv6 + First Hop Security              |
| QoS                    | 8 colas HW, WRR, Strict Priority                        |
| Refrigeración          | Mayoría fanless                                         |
| Montaje                | Rack 19", compacto                                      |
| Garantía               | Garantía limitada de por vida                           |
